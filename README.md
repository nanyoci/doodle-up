# DoodleUp
- An educational interactive story book web app to help children learn English
- This project was created for CZ3002 Advanced Software Engineering
- This folder contains the frontend for DoodleUp

## How to Use
### How to Setup
1. Install Node
2. `npm install`

### How to Run
1. `npm start`

### How to Run Tests
1. `npm test`
